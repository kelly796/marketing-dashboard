# Marketing Dashboard

A comprehensive marketing dashboard built with Netlify Functions and integrated with multiple marketing platforms.

## Integrations

- Meta (Facebook/Instagram)
- Active Campaign (CRM)
- Google Analytics 4
- YouTube Analytics
- Halaxy
- Google Search Console

## Setup

1. Install dependencies: `npm install`
2. Configure environment variables for each service
3. Deploy to Netlify: `netlify deploy`

## Features

- Real-time data aggregation from multiple sources
- Automated daily updates
- Responsive dashboard interface
