How ActiveCampaign Is Structured

List = entry point. Adding a contact to a list triggers their main onboarding sequence automatically.

Tag = which sequence runs. Each tag triggers its own separate automation. A contact can have multiple tags and run multiple automations at the same time — they don't conflict.

Example: A new EP member is added to [HQ] EP Members (fires EP onboarding) and tagged Pass - 2 Week Gym (fires gym pass email) in the same Zap. Both automations run concurrently.

ZAP 4 GymMaster: New Gym Membership → AC

Step 2 — Filter: Membership type contains Perf Base
